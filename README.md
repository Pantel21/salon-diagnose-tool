# Salon Diagnose & Entwicklungs-Framework

Interaktives Web-Tool für die strukturierte Salon-Analyse und Entwicklung im Salonimpuls OS.

## Features

- **6-Phasen-Flow**: Salon-Profil → Datenerhebung → IST-Analyse → SOLL-Definition → Maßnahmenplan → Umsetzungs-Tracking
- **3-Schicht-Datenerhebung**: KI-Datenanreicherung, Beobachtung, Befragung
- **8 Kompetenzfelder**: Strategie, Marketing, Kundenerlebnis, Führung, Wirtschaft, Prozesse, Didaktik, Digitalisierung
- **45 Methoden**: Starter (10) / Growth (+15) / Excellence (+20)
- **Spinnennetz-Diagramm**: IST/SOLL-Vergleich
- **Automatische Gap-Analyse** mit Methodenzuordnung
- **Rollen-Umschaltung**: Berater / Inhaber
- **JSON-Export** der Ergebnisse

## Tech Stack

- React 18 + TypeScript + Tailwind CSS
- Gebündelt als Single-Page HTML (standalone)

## Deployment

Statische Site auf Vercel. `public/index.html` ist das gebündelte Artifact.
